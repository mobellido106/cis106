# What Git
**Git**  Git is a popular version control system.
It was ***created by Linus Torvalds in 2005***, and has been ***maintained by Junio Hamano since then***.

It is used for:
 * Tracking code changes
 * Tracking who made changes
 * Coding collaboration


# What is GitHub? #
 **GitHub** is a web-based platform for version control and collaboration, using **Git** to let developers store, manage, and share code for software projects, offering features like bug tracking, task management, and code review to facilitate teamwork on both public and private projects. It's essentially a central hub where developers can track changes, contribute to shared codebases, and showcase their work, making it crucial for open-source and professional development. 

# What is Slack? #
**Slack** is a cloud-based team communication platform that ***was co-founded by Stewart Butterfield and Meg Matthias***. It originated as an internal communication tool within the company Tiny Speck, which developed the online game Glitch. In 2013, **Slack** was launched publicly and has since become a popular platform for businesses to facilitate communication and collaboration.
